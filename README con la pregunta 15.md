### Pregunta 1
Un algoritmo es un proceso que se sigue para poder realizar una tarea que tengamos que hacer. Los pasos que seguiría para resolver un problema de programación es: 1. leer varias veces la instrucción. 2. Entender que pasos hay que seguir. 3. Arrancar con el código.

### Pregunta 2 
Un puntero almacena dirección de memoria en algún lugar. El operador * es para acceder a un valor y & es para obtener la dirección de memoria

### Pregunta 3
Un arreglo es una estructura que colecciona una cierta cantidad de datos y sus posiciones. para declararlo debes mencionar le tipo de dato y los elementos deben ir en corchete. Para acceder a los elementos de un array, se comienza por el 0. 

### Pregunta 4 
Un alcance sirve ver hasta que punto una de tus variables se puede utilizar. Variable es dentro de un bloque y global es fuera de los bloques. 

### Pregunta 5
